Smart Home Automation System

Designed and developed a smart home control system on the Tiva C microcontroller. Implemented UART communication to bridge embedded hardware with a mobile application, and built feedback-based control logic to dynamically regulate temperature based on user input.

Tiva C UART C Real-Time Control Mobile Integration

The Smart Home Automation System was built on the Texas Instruments Tiva C Series microcontroller (TM4C123GH6PM), leveraging its on-chip UART peripheral to establish a reliable serial communication link between the embedded controller and a companion mobile application. The system processed user commands in real time, parsing incoming data packets to trigger actions such as adjusting thermostat setpoints, controlling relay-driven appliances, and reading analog sensor outputs.

A feedback control loop was implemented in C to continuously monitor temperature sensor data and compare it against user-defined thresholds, automatically actuating a PWM-driven fan and heating element to maintain the target climate. Interrupt-driven UART handling ensured low-latency response times, while careful register-level peripheral configuration demonstrated deep understanding of bare-metal embedded programming without relying on high-level abstraction layers.
